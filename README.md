# Flipkart-Product-Reviews
## Project Overview
This project focuses on analyzing product reviews from Flipkart to understand customer sentiment and product performance.  
It includes text preprocessing, sentiment classification, and visual insights.

## Objectives
- Clean and preprocess review text  
- Perform sentiment analysis using NLP  
- Visualize rating distributions and frequent keywords  

## Tools & Libraries
- Python  
- Pandas, NumPy  
- NLTK / TextBlob  
- Scikit-learn  
- Matplotlib, Seaborn  

## Results
The analysis identifies customer satisfaction trends and provides insights into what users like or dislike about products.
